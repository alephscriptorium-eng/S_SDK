# Junturas

Una juntura es el material que vive *entre* dos holones: no pertenece del todo
a ninguno. Es la grieta del primero que abre paso al segundo — el "ya estaba
latente" que convierte piezas sueltas en holarquía.

Aquí cae el **material excedente** de las fuentes: lo que sobra de Zeus y Cristo
y no es ni Zeus ni Cristo, sino el Logos que los une. De la juntura 01↔02 se
**destila la cuarta pieza** (holón 02).
