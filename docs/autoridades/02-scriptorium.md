# 02 — Scriptorium

**Rol:** Marco del proyecto.
**Fuente:** pub.escrivivir.co/scriptorium

## Concepto (visión del proyecto)
Scriptorium: clave transmedia tras la digitalización (Shannon). El SDK opera generando transmedia a partir de corpus propios.

## Definición documentada
"Puerta de entrada a las salas layer2 del PUB." Workspace colaborativo: herramienta + deliberación.

## Estructura
- **Hackería** — taller: herramientas, plugins, SDKs, agentes.
- **Parlament** — gobernanza layer2 y arquitectura de producción.
- **Future Rooms** — sesiones, indexado de corpus, materiales revisitados.

## Rasgos técnicos
- "Futures-engine": ramificación dramatúrgica.
- Layer2 implementado como LARP (Live Action Role-Playing).
- VPS con federación Node-RED, almacenamiento persistente, mesh P2P (HTTPS/WSS), endpoints (admin, rooms, MCP, npm registry).
